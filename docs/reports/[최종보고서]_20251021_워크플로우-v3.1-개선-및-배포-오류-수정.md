#***REMOVED***🔧 **배포 오류 수정**

##***REMOVED*****wrangler.toml TOML 파싱 오류**

**오류 메시지:**
```
ParseError: Unterminated inline array
at line 21, column 8: vars = {
```

**원인:**
```toml
[env.production]
vars = {                                    ← 잘못된 인라인 배열 형식
  BACKEND_RPC_URL = "https://...",
  ...
}
```

**수정:**
```toml
[env.production]

[env.production.vars]              ← TOML 테이블 형식으로 변경
BACKEND_RPC_URL = "https://..."
TON_RPC_HTTPS_ENDPOINT = "https://..."
NETWORK_FEE_TON = "0.03"
```

✅ **수정 완료** (커밋: 9c49c49)

##***REMOVED*****1️⃣ Lite 워크플로우 추가 (빠른 개발용)**

**목표**: Full 워크플로우의 무거움 해결

**Lite 워크플로우 (4단계)**
```
[Step 1] 요청 분석 (빠르게)
  → ssot 깊이 확인 SKIP
  → 즉시 작업 시작

[Step 2] 코드 수정 (단독)
  → 문서 동기화 SKIP
  → 코드만 완성도 있게

[Step 3] Git 커밋
  → "fix: [설명]" or "docs: [설명]"
  → 버전 변경 선택사항

[Step 4] 배포 및 칸반
  → git push
  → kanban [Done]
```

**언제 사용?**
- 📝 문서만 수정
- 🐛 버그 수정 (코드 1-3줄)
- ⚡ 빠른 반복
- 🔍 리팩토링 (기능 변경 없음)

**효과**: 작업 시간 **50% 단축**
