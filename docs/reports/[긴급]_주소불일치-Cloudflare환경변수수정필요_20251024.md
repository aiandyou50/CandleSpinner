## 💡 추가 설명: UQ vs EQ 주소

```
TON 주소는 2가지 형식:

1. Bounceable (EQ...) - 기본 형식
   ├─ 트랜잭션 실패 시 반송됨
   └─ 스마트컨트랙트 사용 시 권장

2. Non-Bounceable (UQ...) - 보안 형식
   ├─ 트랜잭션 실패 시 반송 안 됨
   └─ 일반 지갑 사용 시 권장

⚠️ 중요: 둘 다 같은 지갑을 가리킴 (접두사만 다름)
   하지만 끝부분도 달라서 이상함
   
실제 문제: 주소 자체가 다른 지갑
```
