    amount: result.tonAmount,
    payload: result.boc
  }]

#***REMOVED***2️⃣ RPC 방식 오류 ❌

##***REMOVED***오류 메시지
```
API 응답 상태: 500
❌ 오류: RPC Error -32079: Origin not allowed
```

##***REMOVED***원인 분석
| 항목 | 내용 |
|------|------|
| 오류 코드 | -32079 |
| 오류 유형 | CORS/Origin 검증 |
| 위치 | 백엔드 (클라우드플레어 워커) |
| 원인 | Ankr RPC가 클라우드플레어 Origin 미등록 |

##***REMOVED***해결 방안

###***REMOVED***방안 A: 환경 변수 확인 (현재)
```
ANKR_JSON_RPC_HTTPS_ENDPOINT 
설정 값 확인 필요
```

**확인 사항**:
- [ ] 올바른 Ankr RPC URL인가?
- [ ] Ankr 대시보드에서 Origin 등록되었나?
- [ ] 네트워크 연결 확인

###***REMOVED***방안 B: 다른 RPC 제공자로 변경 (대체)
```typescript
// 예시 - 다른 RPC 제공자
const rpcUrl = 'https://toncenter.com/api/v2/jsonRPC';
// 또는
const rpcUrl = 'https://toncenterapi.com/api/v2/send';
```

**가능한 대체 RPC**:
- Toncenter (`https://toncenter.com`)
- Getblock.io
- QuickNode

###***REMOVED***방안 C: 프록시 구현
```typescript

// /api/rpc-proxy 생성

  return rpcResponse;
}
```
