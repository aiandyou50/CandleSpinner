## 🎯 시스템 흐름

```
1. 사용자가 게임에서 크레딧 획득
2. 사용자가 인출 요청 (프론트엔드)
   ↓
3. 백엔드가 서명된 바우처 발급 (오프체인, RPC 0회)

4. 프론트엔드가 바우처를 컨트랙트에 제출 (사용자 가스비)
   ↓
5. 컨트랙트가 서명 검증 후 CSPIN 전송
   ↓
6. 사용자 지갑에 CSPIN 도착
```
